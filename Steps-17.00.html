Creating the matrix visual 17

  1. Drag and drop the matrix from the visualization pane
  2. Place the matrix in lower right of the report
  3. Drag and drop the following fields
    Rows : Started Mths BK
            SMIC_Date Added_UserID_Ps-CustomerID
   Columns : Last Payment Mths
   Values : User ID
   4. Select the format  icon
    Format:
   -General
    Properties: Size= Height 390, Width 940, Position= Horizontal 335, Vertical 330
    Title ON=Last Payment Mths
    Font = Segoe UI 10
    Tooltip ON
  -Visual
    Values: Font =Segoe UI 12
    Column headers: Font = Segoe UI 10
    Row headers: Font = Segoe UI 10, +/-Icons ON
    Column Subtotal ON
    Row Subtotal ON
    Cell elements:
    Background color ON, fx->fields value-> select the "color" measure->OK
    Web URL ON, fx->fields value-> select the "color" measure->OK
    visual border ON
    Drag and drop a text box on the top of the matrix
    Format:
  -General
    Properties: size = Height 22, Width 940
    Position = Horizontal 335, Vertical 312
    Title ON: Payments Track
    Font = Segoe UI 14
    Text color = White
    Background color "#12239E"
    Visual border ON
