Create Additional Table 11
-Measure table
  1. Go to the "Home" tab, click "Enter Data"
  2. Rename the table Measure Table, click "Load"


-Last Refreshed Date Table
  1. On Power BI Desktop ribbon, Click on Transform data
  2. Click on New Source button, select Blank query from the list and click on Connect
  3. Under properties in the Name box type in 'Last Refreshed Date'
  4. Type the Dax formula ' =DateTime.LocalNow()'
  5. Click on "Transform"
  6. Click on "Convert to Table" button and rename the column “Last Refreshed Date” and Click  on "Close & apply" button 
    -Create a table to sort the matrix by date later
  1. Go to the "Home" tab, click "Transform Data"
  2. Right click and duplicate the table
  3. Rename the new table "Sort _vw_ADF_PaySimple_ISA_Reporting(2)"
  4. Select the UserID, Started Mths BK, Date Added, Last payment Mth
  5. Click" remove other columns"
  6. Close and apply



-Create an index column named Sort_Dateadded_UserID in the second table
  1. Go to the "Home" tab, click "Transform Data"
  2. Select "Add column"->Index column->"From 1" and Rename the column"Sort_Dateadded_UserID"
  3. Click "close and apply" to save
    The same process should be followed in order to sort the Last Payment Mths column      in the main table.
