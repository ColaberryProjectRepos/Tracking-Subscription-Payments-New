Creating the Tooltip page that will be used in the matrix 18

Create the table
  1.  Create a new blank page.
  2. Select The Format Icon and then the Page information.
  3. Turn Allow use as tooltip ON.
  4. Select Canvas settings and then Type go to custom Height 400 and Width 1025 close it.
  5. Select the table visual in the visualization field and make it to fit into the custom canvas.
  6. Now you can drag and drop the column name that you need to show in the tooltip in the Columns field Such as ( UserID, PaymentsRemaining_Cnt, Started_Mths, etc.,)


Format the table column SMIC_NextPaymentDate  font color to turn RED when the date < TODAY 
  1. Select the table and click on Format Visual Icon.
  2. Select Cell elements and then Series (chose the column SMIC_NextPaymentDate)
  3. Turn Font color ON then click on the fx (conditional formatting) it will take you to the Front color page.
    Format style: Field Value
    Base the format style on the measure ColorNextPaymentDate then click OK.
