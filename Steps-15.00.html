Adding Cards Visual 15

-Total Subscribers card
     For the main KPI
  1. Select card in the visualizations pane
  2. Drag and drop the total subscribers measure in the field.
  3. Select the format icon
   Visual: Callout value=Font Segoe UI 20, category label OFF
   General: 
   Property = Size: Height 140, Width 337, Title ON then enter the title name in the text       field, Font Segoe UI 14, Text color White, Background color #12239E, 
   visual border =ON , color #B3B3B3
   Horizontal alignment: Center
For the sub KPI
  1. Select the card in the visualizations pane
  2. Drag and drop the %SMI Total Subscribers measure in the field.
  3. Select the format  icon   
    Visual: Callout value=Font Segoe UI 14, category label OFF
    General: 
    Property = Size: Height 50, Width 100
    Tooltips ON t
    Drag and drop this card into the TOTAL SUBSCRIBERS card.
   The same process will be repeated for all main KPI, sub KPI and Last refreshed date;          All  you have to do is to change the fields name.
