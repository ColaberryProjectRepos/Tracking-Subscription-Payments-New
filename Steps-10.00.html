Cleaning and Transforming the table 10

 Data transformations:


In this step, we are going to clean and transform the data
    -Convert Started_MthsBk and LastPayment_Mths columns  to INT
  1. Go to the "Home" tab, click "Transform Data"
  2. Select the column name and chose "Whole number" in the drop down list of "Data Type"
  3. Click close & Apply to save the changes
    -Change data type  SMIC_Date added column  to Date
  1. Go to the "Home" tab, click "Transform Data"
  2. Select the column name and chose "Whole number" in the drop down list of "Data Type"
  3. Click close & Apply to save the changes
    -Categorize Data Link column as "Web URL"
  1. Click on the Data Link column
  2. Go to the Column tool tab
  3. In the Data Category drop down, select "Web URL"
    - Create a new column to sort the Last Payment month
  1. Go to the "Home" tab, click "Transform Data"
  2. Select Last Payment Mth column
  3. Click "Add column"
  4. Select "Column from example"(From selection)
  5. Click OK
